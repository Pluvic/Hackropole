# Crayon Cochon

## Spécification

**Catégorie**: Cryptographie

**Difficulté estimée**: Intro

**Année**: 2019

## Description

Il s'agit d'une image avec un genre d'encodage avec des symboles.

## Write-Up

Il s'agit d'un encodage de type "pigpen cipher". En utilisant la table de correspondance entre les symboles et les lettres, on peut décoder le message pour obtenir le flag.
